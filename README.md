## Tentang Proyek Ini

Aplikasi Portal Berita berbasis Laravel.  
Admin dapat menambah, mengedit, dan menghapus berita, kategori, serta komentar.  
Pengguna dapat melihat berita dan menulis komentar.

### Fitur
- Halaman Admin
  - Login Admin
  - Kelola Berita (CRUD)
  - Kelola Kategori (CRUD)
  - Hapus Komentar
- Halaman Pengguna
  - Lihat Berita
  - Tulis Komentar
